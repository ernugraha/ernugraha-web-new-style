## Halo, selamat datang!

Ini adalah template Jekyll dengan desain Apple-style yang bersih dan modern. Dirancang untuk performa tinggi dan kemudahan kustomisasi.

### Fitur Utama

- **Navbar dinamis** — transparan di atas, blur frosted-glass saat scroll
- **Dark mode** — beralih tema dengan satu klik, tersimpan otomatis
- **Pencarian bawaan** — cari semua konten secara instan dengan `Ctrl+K`
- **search_exclude** — tandai halaman agar tidak muncul di hasil pencarian
- **Responsif penuh** — tampil sempurna di semua ukuran layar

### Cara Pakai

Edit `_config.yml` untuk menyesuaikan nama situs, deskripsi, dan pengaturan lainnya. Tambahkan post baru di folder `_posts/` dengan format nama file `YYYY-MM-DD-judul-post.md`.

Untuk mengecualikan halaman dari pencarian, tambahkan `search_exclude: true` di front matter.
